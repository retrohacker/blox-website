MANIFESTO

Data should be a lot of things, and data means different things to different people, but we believe all of your files should adhere to the following four tenants:

Data should be
PERSISTANT

Once a file is saved, you should have a strong guarantee that it is safe. Irrespective of hardware failure, your file should be able to be recovered in the future.

Data should be
VERSIONED

Files change over time, and often those changes hold more information than the file itself. You should have access to the historical copies of your files, not just the latest version.

Data should be
INDEPENDENT

No matter what device you save your file to, it should be accessible anywhere. You should be able to access your data from anywhere.

Data should be
PRIVATE

Nobody should be able to access your data other than you, no matter who they are.
